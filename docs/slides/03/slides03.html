Aesthetic Mapping

Visual properties that the data is mapped to

  • x → body mass

  • y → flipper length

  • color → species

Geometric Object

How the data are represented

  • individual penguins → points

The basic ggplot template:

ggplot(data = <dataset_name>) + 
  <geom_function>(mapping = aes(<mappings>))
  • data: data =
  • aesthetic: mapping = aes(...)
  • geometry: geom_function
  • Note: aesthetic mappings can also appear in the ggplot function - “global” - passed down to all subsequent geom layers of the plot

Examples

ggplot(penguins, aes(x = body_mass_g, y = flipper_length_mm, col = species)) + 
  geom_point()
# Compare above plot to plot that would result from code below:
#ggplot(penguins) + 
#  geom_point(aes(x = body_mass_g, y = flipper_length_mm, col = species))

ggplot(penguins, aes(x = body_mass_g)) + 
  geom_histogram()
